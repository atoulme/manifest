= Manifest

* http://manifest.rubyforge.org

== DESCRIPTION:

Manifest is a library to read Manifest files located in jars and OSGi bundles.

== REQUIREMENTS:

Development:
* rspec

== INSTALL:

* sudo gem install manifest
